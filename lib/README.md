# Flutter App Release Demo

This project demonstrates a complete Flutter Android release pipeline.

## Features
- Notes app (add/delete)
- Release build setup
- Keystore signing
- Icons & splash screen
- Permissions & privacy
- Internal testing
- Release documentation

---

## Setup
flutter pub get

---

## Build
flutter build appbundle
